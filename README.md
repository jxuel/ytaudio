# ytaudio
Only play audio form all videos in a playlist.

## Dependencies
 * Qt 5.14.2
 * C++ 11
 * python 3.7+
    * requests

## Build

  ```
your_paht/ytaudio$mkdir build && cd build
your_paht/ytaudio/build$cmake ..
your_paht/ytaudio/build$make
  ```

## Run
Open exeutable file `ytaudio`
